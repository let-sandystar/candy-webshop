🍬 Candy Queen
A modern candy webshop built with Vite and TypeScript

📌 Overview
Candy Queen is a responsive webshop application where users can browse candy products, view detailed product pages, check stock levels, add items to their cart, and complete a full checkout flow. The project was originally developed as a group assignment using agile methodologies such as Scrum and Kanban. This version represents my continued work and refinement of the application.

✨ Features
- Product listing with images, prices, and discount indicators
- Individual product pages
- Real‑time stock level display
- Discounted product handling
- Shopping cart with quantity management
- Checkout flow with order confirmation
- API integrations for products, stock, and orders

💜 My Contributions
I focused primarily on the backend‑connected logic and data flow within the application, including:
- API integrations for fetching products, stock levels, and posting orders
- Order confirmation logic, including generating and displaying confirmation details
- Stock level handling, ensuring accurate updates and preventing overselling
These contributions strengthened the reliability of the checkout process and improved the overall user experience.

🛠️ Tech Stack
- Vite
- TypeScript
- REST API integrations
- SCSS/CSS/HTML

🤝 Agile Workflow
The team worked using agile principles, including:
- Scrum for sprint planning and iterative development
- Regular stand‑ups, retrospectives, and collaborative problem‑solving
- Kanban for visualizing workflow and managing tasks
